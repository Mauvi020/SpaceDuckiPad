🚀 SpaceDuckiPad

A compact macro pad powered by an ESP32-S3.

⸻

🔧 Components

1. ESP32 s3 
2. Encoder
3. 1,3 inch lcd display
4. 9x MX switches 
5. jumper or 30awg wier
6. case (3d printed)
7. key caps (3d printed optional)
